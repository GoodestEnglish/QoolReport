# QoolReport
以前我還在Qool.Dev 開伺服器時候的一個舉報系統插件, 就很簡單一個舉報插件
一年前創作的插件, 最近在整理我的IDE 插件源碼的時候, 發現了這陳舊的插件
其實插件不是寫得太差, 只是功能比較少, 而且BungeeCord 用家可能不適合使用
但無論如何, 希望你們喜歡這個插件 <3

我使用的API 有一個叫GoodestSpigot, 那個其實就是Spigot 核心, 但我的版本只是經過修改和優化而已
